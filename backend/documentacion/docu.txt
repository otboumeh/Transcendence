- /api/users.php:

    POST = crear un nuevo usuario
    formato JSON = ({
        "name": 'name',
        "email": 'email',
        "password": 'password'
    });

    GET = obtener la lista de todos los usuarios
    no tiene formato JSON (body vacio)
    
    GET [/api/users.php?id='id'] = obtener los datos del usuario en cuestion [id]
    no tiene formato JSON (body vacio)

    PATCH [/api/users.php?id='id'] = editar los datos de un usuario
    - formato JSON = ({
        "name": 'new_name',
        "email": 'new_email'
    }); [ esto para editar el username / correo electronico]

    - formato JSON = ({
        "password": 'new_password'
    }) [ este cambia la contrasenha de un user]

    DELETE [/api/users.php?id='id'] = borrar un usuario
    no tiene formato JSON  (con el id basta)

- /api/friend-request.php:

    POST = enviar una solicitud de amistad
    formato JSON = ({
        "sender_id": 'sender_id',
        "receiver_id": 'receiver_id'
    });

    GET [/api/friend-request.php?id='id'] = consultar solicitudes de amistad entrantes
    no tiene JSON pues se comprueba solo con el id

    PATCH = aceptar/declinar una solicitud de amistad
    formato JSON = ({
        "sender_id": 'sender_id',
        "receiver_id": 'receiver_id',
        "action": 'accept/decline'
    });

- /api/friends.php:

    GET [/api/friends.php?id='id'] = ver la lista de amigos del id
    no lleva body JSON ya que se consulta unicamente con el id

    DELETE = eliminar a un amigo
    formato JSON = ({
        "user_id": 'user_id',
        "friend_id": 'friend_id
    })

- /api/ladder.php:

    GET [/api/ladder.php?id='id'] = retorna el ranking de los amigos
    del usuario solicitado

    GET [/api/ladder.php] = retorna el ranking de todos los usuarios de 
    la base de datos [global rank]

- /api/matches.php:

    GET [/api/matches.php?id='id'] = retorna el id del jugador con el ELO mas cercano
    a si mismo para organizar las partidas balanceadas

    GET [/api/matches.php?id='id'] (con body) = retorna el numero de jugadores pedido con un elo parecido
    formato JSON = ({
        "creator_id": 'creator_id',
        "number": 'numero' // aqui va el numero de personas solicitadas para el torneo para que sea variable
    })

    POST [/api/matches.php] = actualiza el elo despues de un partido
    formato JSON = ({
        "winner_id": 'winnerId',
        "loser_id": 'loserId'
    })
